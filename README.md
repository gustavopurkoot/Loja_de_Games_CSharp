# 🎮 Loja de Games - C#

Sistema de gerenciamento de uma loja de games desenvolvido em C#, com foco em conceitos de programação orientada a objetos (POO), como classes, métodos e manipulação de dados.

## 📌 Sobre o projeto

Este projeto simula o funcionamento básico de uma loja de jogos, permitindo gerenciar produtos, controlar estoque e realizar operações relacionadas a vendas.

Foi desenvolvido com o objetivo de praticar conceitos fundamentais de C# e lógica de programação.

## 🚀 Funcionalidades:

- 📦 Cadastro de jogos
- 💰 Definição de preços
- 🎮 Identificação de plataforma (console)
- 🔢 Controle de quantidade em estoque
- 📊 Cálculo do valor total em estoque
- 🧾 Exibição das informações dos produtos

## 🛠️ Tecnologias utilizadas:

- C#
- .NET
- Programação Orientada a Objetos (POO)

## 🧠 Conceitos aplicados:

- Classes e Objetos
- Construtores
- Métodos
- Encapsulamento
- Listas (List<T>)
- Entrada e saída de dados (Console)
